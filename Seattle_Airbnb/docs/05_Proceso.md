Proceso de análisis

1. Descarga de datos desde Airbnb Seattle.
2. Análisis exploratorio inicial para entender estructura y calidad.
3. Diseño y creación del modelo de base de datos en SQL.
4. Limpieza y transformación de datos.
5. Análisis en SQL sobre datos limpios.
6. Carga y modelado en Power BI.
7. Desarrollo de dashboards e insights.
8. Documentación final del proyecto.
