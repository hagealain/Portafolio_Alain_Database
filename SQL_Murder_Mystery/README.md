### The SQL Murder Mystery — SQL Ejercicio de práctica by Alain Zahid Tinoco Rodríguez

## Descripción

Este repositorio contiene mi resolución personal de The SQL Murder Mystery, un reto público de SQL diseñado para practicar la resolución de problemas mediante la exploración y análisis de datos en una base de datos relacional.

El objetivo del ejercicio es identificar al responsable de un crimen ficticio utilizando consultas SQL, poniendo énfasis en el razonamiento analítico y la toma de decisiones basada en datos.

## Objetivo del ejercicio

Este ejercicio tiene como finalidad demostrar:

- Pensamiento analítico aplicado a datos.

- Exploración estructurada de una base de datos.

- Uso de SQL para investigar escenarios abiertos.

- Capacidad para formular y validar hipótesis.

El foco no está únicamente en la respuesta final, sino en el proceso seguido para llegar a ella.

## Enfoque de resolución

La investigación se abordó de manera progresiva:

- Exploración inicial para comprender el modelo de datos.

- Identificación de información clave relacionada con el evento.

- Formulación de hipótesis basadas en los datos disponibles.

- Refinamiento de resultados mediante filtros y relaciones entre tablas.

- Validación de hallazgos hasta llegar a una conclusión consistente.

El detalle del razonamiento y las decisiones tomadas se documenta en "docs/investigation_notes.md".

## Tecnologías utilizadas

- SQL (consultas, filtros, joins).

- Base de datos relacional proporcionada para el reto por el equipo de Knight Lab.

- No se utilizaron dashboards ni herramientas de visualización, ya que el ejercicio está enfocado exclusivamente en análisis y razonamiento con SQL.
